## Installation

You can install _jaxscat_ via [pip](https://pip.pypa.io/) from [PyPI](https://pypi.org/project/jaxscat/) or from source.

```console
$ pip install jaxscat
```

```console
$ pip install git+https://github.com/matt-black/jaxscat.git
```
